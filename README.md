# Setup
To run this project, install it locally using npm.
```
$ Clone Repo "https://github.com/Alisa1989/budget-app.git"
```
Frontend .env include:
```
PORT=
```
Backend .env include:
```
MONGODB_CONNECT_STRING=
PORT=
NODE_ENV = development
ACCESS_TOKEN_SECRET =
```
Terminal window 1
```
$ cd server
$ npm install
$ npm run dev
```
Terminal window 2
```
$ cd client
$ npm install
$ npm start
```
